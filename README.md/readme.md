# Dynamic Quiz App 

A dynamic quiz web app with category & difficulty selection, 30s timer per question, and result analytics with charts.
Automation testing is done using Selenium (Python).

## Features
- Category & Difficulty selection
- 30 seconds timer per question
- Result page with score + correct/incorrect/skipped
- Detailed review + charts (Chart.js)
- Selenium automation with screenshots
